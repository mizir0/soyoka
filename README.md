# 自然派・無添加スキンケアブランド LP（架空案件）

## ■ 概要
敏感肌に悩む30代女性をターゲットにした、
自然派・無添加スキンケアブランドのランディングページ制作。

企画設計からデザイン、HTML/CSSコーディングまで一貫して制作しました。

---

## ■ ペルソナ

森下 香澄（34歳）
宮城県仙台市在住／保険会社営業  
夫・小2の息子・年中の娘の4人家族。

出産後に敏感肌になり、化学成分入りスキンケアで赤みや刺激を感じるように。
仕事と育児の両立でストレスを抱え、肌荒れに悩んでいる。

「せめて肌だけはきれいでいたい」
と考え、無添加で自然素材を使用した安心できるスキンケアを求めている。

---

## ■ 制作目的

・敏感肌でも安心して使えるブランドイメージの確立  
・安心感・やさしさ・癒しを伝える世界観構築  
・スマホユーザーを想定した導線設計  

---

## ■ 設計意図

### 配色
ベージュやくすみグリーンを基調とし、
「自然」「安心感」「やさしさ」を表現。

### レイアウト
余白を広めに設計し、
忙しい日常の中でも“呼吸できる”ような印象を意識。

### フォント
女性らしさと信頼感を両立する書体を選定。

### 導線設計
スマートフォン閲覧を想定し、
3スクロール以内にCTA（購入ボタン）を配置。
