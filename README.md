## 🎯 Project Purpose

- 敏感肌でも安心して使えるブランドイメージの確立
- 自然・無添加・やさしさを伝える世界観設計
- スマホユーザーを想定したCV導線設計


---


## 🎨 Design Concept


### 🌿 Color
ベージュ・くすみグリーンを基調に  
「安心感」「自然」「癒し」を表現。


### ✨ Layout
余白を広く取り、  
忙しい日常の中でも“呼吸できる”印象を設計。


### 📱 Responsive
スマートフォン閲覧を想定し  
3スクロール以内にCTAを配置。
